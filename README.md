# Sales Performance Dashboard

## Project Overview

This project analyzes sales performance data using SQL and R. The objective was to identify sales trends, evaluate regional performance, compare product categories, and determine high-value customers through data-driven analysis and visualization.

## Tools Used

* MySQL Workbench
* SQL
* R
* RStudio

## Dataset

Sales transaction dataset containing order details, customer information, product categories, regions, and sales values.

## Key Analysis

1. Sales by Category
2. Sales by Region
3. Monthly Sales Trend
4. Top Customers by Sales

## Project Workflow

CSV Dataset → MySQL → SQL Analysis → Views → CSV Export → R Visualization

## Visualizations

* Sales by Category
* Sales by Region
* Monthly Sales Trend
* Top Customers by Sales
